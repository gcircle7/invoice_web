# 노션 기반 견적서 공유 서비스 MVP PRD

## 🎯 핵심 정보

- **목적**: 노션 데이터베이스에 입력한 견적서를 클라이언트가 고유 링크로 즉시 확인하고 PDF로 수령할 수 있도록 하여, 별도 견적서 툴 없이 견적 공유 업무를 자동화한다.
- **타겟 사용자**: 클라이언트에게 견적서를 발송해야 하는 1인 프리랜서 및 소규모 개발자/디자이너.

---

## 🚶 사용자 여정

### 전체 플로우 다이어그램

```
[작성자(나) 플로우]

노션 데이터베이스에 견적서 입력
        │
        ▼
견적서 목록 페이지 접속
        │
        ├─── 견적서 항목 클릭
        │           │
        │           ▼
        │    견적서 상세 페이지 (미리보기)
        │           │
        │           ▼
        │    고유 링크 복사 → 클라이언트에게 전달
        │
        └─── (추후 목록에서 관리 계속)


[클라이언트 플로우]

작성자로부터 고유 링크 수신
        │
        ▼
견적서 공개 뷰어 페이지 접속
        │
        ├─── 견적서 내용 확인
        │
        └─── PDF 다운로드 버튼 클릭
                    │
                    ▼
             PDF 파일 저장 완료
```

### 페이지 간 전환 조건

| 출발 페이지             | 도착 페이지             | 전환 조건                      |
| ----------------------- | ----------------------- | ------------------------------ |
| 견적서 목록 페이지      | 견적서 상세 페이지      | 목록에서 특정 견적서 항목 클릭 |
| 견적서 상세 페이지      | 견적서 목록 페이지      | 뒤로가기 또는 목록 버튼 클릭   |
| 외부 링크(이메일 등)    | 견적서 공개 뷰어 페이지 | 고유 링크 접속                 |
| 견적서 공개 뷰어 페이지 | (종료)                  | PDF 다운로드 후 이탈           |

### 사용자 선택 분기점

- **견적서 목록 페이지**: 기존 견적서 클릭(상세 이동) 또는 노션에서 새 견적서 작성 후 새로고침
- **견적서 공개 뷰어 페이지**: 내용 확인만 하거나 PDF 다운로드 선택

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명                  | 설명                                                                       | MVP 필수 이유              | 관련 페이지             |
| -------- | ----------------------- | -------------------------------------------------------------------------- | -------------------------- | ----------------------- |
| **F001** | 노션 견적서 목록 조회   | 노션 데이터베이스에서 전체 견적서 목록을 가져와 표시                       | 핵심 데이터 소스 연동      | 견적서 목록 페이지      |
| **F002** | 견적서 상세 조회        | 노션에서 특정 견적서의 모든 항목 데이터를 가져와 표시                      | 견적서 내용 확인 핵심 기능 | 견적서 상세 페이지      |
| **F003** | 고유 링크 생성 및 복사  | 견적서별 고유 슬러그(노션 페이지 ID 기반)로 공개 URL 생성 및 클립보드 복사 | 클라이언트 공유 핵심 수단  | 견적서 상세 페이지      |
| **F004** | 견적서 공개 뷰어 렌더링 | 고유 링크로 접속 시 견적서를 웹 페이지로 렌더링                            | 클라이언트 열람 핵심 기능  | 견적서 공개 뷰어 페이지 |
| **F005** | PDF 다운로드            | 렌더링된 견적서를 PDF 파일로 변환하여 다운로드                             | 클라이언트 문서 수령 수단  | 견적서 공개 뷰어 페이지 |

### 2. MVP 필수 지원 기능

| ID       | 기능명               | 설명                                                        | MVP 필수 이유                  | 관련 페이지                            |
| -------- | -------------------- | ----------------------------------------------------------- | ------------------------------ | -------------------------------------- |
| **F006** | 견적서 상태 표시     | 견적서의 상태(초안/발송됨/승인됨 등)를 목록과 상세에서 표시 | 견적서 관리 최소 식별 수단     | 견적서 목록 페이지, 견적서 상세 페이지 |

### 3. Phase 5 고도화 기능

| ID       | 기능명                  | 설명                                                                            | 관련 페이지                            |
| -------- | ----------------------- | ------------------------------------------------------------------------------- | -------------------------------------- |
| **F008** | 다크/라이트 모드 토글   | 사용자가 선호하는 테마(라이트/다크/시스템)를 선택하고 localStorage에 영속 저장  | 전체 페이지                            |
| **F009** | 목록 페이지 링크 복사   | 견적서 목록 테이블에서 행 이탈 없이 바로 공유 링크를 복사할 수 있는 버튼 추가   | 견적서 목록 페이지                     |
| **F010** | 관리자 전용 레이아웃    | 좌측 사이드바 + 우측 콘텐츠 2단 구조의 관리자 대시보드 레이아웃 적용            | 견적서 목록 페이지, 견적서 상세 페이지 |

### 4. MVP 이후 기능 (제외)

- 클라이언트 인증 및 접근 제어 (비밀번호 보호 링크)
- 견적서 승인/거절 인터랙션 (클라이언트 서명)
- 이메일 자동 발송
- 견적서 버전 관리
- 통계 대시보드 (열람 횟수, 전환율)

## 🗄️ 데이터 모델

> 별도 DB 없이 노션 데이터베이스를 단일 데이터 소스로 활용합니다.

### Invoice (견적서) — 노션 데이터베이스 페이지

| 필드         | 설명                                          | 타입/관계                   |
| ------------ | --------------------------------------------- | --------------------------- |
| id           | 노션 페이지 고유 ID (공개 링크 슬러그로 활용) | UUID                        |
| title        | 견적서 제목                                   | Text                        |
| client_name  | 클라이언트(수신자) 이름 또는 회사명           | Text                        |
| status       | 견적서 상태                                   | Select (초안/발송됨/승인됨) |
| issue_date   | 발행일                                        | Date                        |
| expiry_date  | 유효기간 만료일                               | Date                        |
| total_amount | 견적 총액                                     | Number                      |

### InvoiceItem (견적 항목) — 노션 하위 데이터베이스 또는 테이블 블록

| 필드       | 설명               | 타입/관계    |
| ---------- | ------------------ | ------------ |
| id         | 항목 고유 ID       | UUID         |
| invoice_id | 상위 견적서 ID     | → Invoice.id |
| item_name  | 품목명             | Text         |
| quantity   | 수량               | Number       |
| unit_price | 단가               | Number       |
| amount     | 금액 (수량 × 단가) | Number       |
